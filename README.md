# ⚽ Football Player Card Website

A simple and responsive football player card website built using **HTML** and **CSS**.  
This project displays stylish football player profile cards including images, positions, and statistics.

---

## 📌 Features

- 🖼️ Player image display  
- 🧾 Player name and position  
- 📊 Player statistics (goals, assists, matches, etc.)  
- 🎨 Modern card UI design  
- 📱 Fully responsive layout  
- ✨ Smooth hover effects  

---

## 🛠️ Technologies Used

- **HTML5** – Website structure  
- **CSS3** – Styling and layout design
